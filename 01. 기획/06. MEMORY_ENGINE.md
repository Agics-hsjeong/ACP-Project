# 1. Purpose

Memory Engine은 AI Character Playground의 핵심 시스템이다.

목표는 다음과 같다.

- 기억을 장기적으로 유지한다.
- 캐릭터가 사용자를 기억하게 만든다.
- 관계를 지속적으로 성장시킨다.
- 캐릭터 붕괴를 방지한다.
- 몰입감을 극대화한다.
