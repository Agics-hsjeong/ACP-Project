# API_SPEC.md

# AI Character Playground API Specification

Version: 1.0

Base URL

text /api/v1 

Authentication

text JWT Bearer Token 

---

# 1. Authentication

---

## Login

### POST

http POST /auth/login 

### Request

json {   "email": "user@example.com",   "password": "password123" } 

### Response

json {   "access_token": "...",   "refresh_token": "...",   "user": {     "id": "uuid",     "nickname": "휘선"   } } 

---

## Refresh Token

### POST

http POST /auth/refresh 

### Request

json {   "refresh_token": "..." } 

### Response

json {   "access_token": "..." } 

---

## Logout

### POST

http POST /auth/logout 

---

# 4. Character

---

## Create Character

### POST

http POST /characters 

### Request

json {   "world_id": "uuid",   "name": "엘리아",   "description": "...",   "core_identity": {},   "persona_traits": {},   "speech_style": {} } 

---

## Character List

### GET

http GET /characters 

### Query

text ?search=엘리아 ?genre=Fantasy ?tag=Tsundere ?page=1 

---

## Character Detail

### GET

http GET /characters/{character_id} 

### Response
