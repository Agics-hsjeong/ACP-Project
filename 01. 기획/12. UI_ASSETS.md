# 5. 작업 규칙

## 추가·수정 시

1. **Mockup** → `mockup/`에만 추가. 파일명: `Screen NN - {화면명}.png`
2. **Pack** → `pack/`에만 추가. 파일명: `NN. {category}_pack.png`
3. `figma.config.json`의 `screens` / `builtScreens` 갱신
4. 본 문서 §2·§3 테이블 갱신
5. Phase 변경 시 `00. PROJECT_ALIGNMENT.md` §8 동기화

## 금지

- Mockup을 `pack/`에, Pack을 `mockup/`에 넣지 않는다
- 루트 `02. UI 디자인/`에 PNG를 직접 두지 않는다 (하위 폴더만 사용)
- Pack PNG 텍스트를 구현 코드에 하드코딩하지 않는다 (토큰·i18n 사용)
