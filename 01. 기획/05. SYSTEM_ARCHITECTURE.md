# 1. Architecture Overview

AI Character Playground는 **Living World Engine**을 핵심으로, AI 추론과 일반 웹 서비스 기능을 분리한 이벤트 기반 아키텍처를 사용한다.

주요 목표는 다음과 같다.

- 빠른 응답 속도
- 장기 기억 유지
- 캐릭터 일관성 보장
- 높은 확장성
- AI 서비스 독립성
